# Structa v0.8 — Tile Matrix Native Spec

## Goal
Reach the reference aesthetic: modular, geometric, color-blocked, calm, premium, and strictly native to Rabbit R1.

## Non-negotiables
- Exact R1-first screen profile
- Portrait-safe square composition
- No desktop chrome feel
- No vague decorative animation
- One primary action only
- Tiny, quiet log
- High contrast and strong geometry

## Visual strategy
- Replace graph-like UI with a modular tile composition
- Use 6 square tiles as the core system view
- Each tile carries one strong geometric motif
- The composition should feel like a curated poster, not a dashboard
- Lines and shapes must be solid, readable, and deliberate

## Layout rules
- Top bar: minimal title and status only
- Main field: 2x3 tile matrix, centered
- Action button: one pill, centered near bottom of main field
- Log: 3 to 5 short lines, tightly spaced

## Motion rules
- Use only subtle transform and reveal animations
- No particles
- No blur-heavy effects
- No chaotic motion
- Cascade should feel like a sequence of precise activations

## Icon / motif rules
- Use abstract solid forms over thin line icons
- Prefer circles, squares, triangles, diamonds, rings, and crosses
- Motifs should be high-contrast and tile-native
- Keep all symbols large and readable at a glance

## UX rules
- One thumb use
- Fast comprehension
- Minimal text
- Quiet feedback
- No extra UI chrome
- Nothing that feels like a browser app

## Success criteria
- Feels like it belongs on the R1 hardware
- Looks like a refined geometric object, not a web page
- Feels premium, restrained, and memorable
- The reference aesthetic is recognizable in the composition and rhythm
