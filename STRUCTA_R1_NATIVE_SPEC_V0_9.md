# Structa v0.9 — Composition Native Spec

## Goal
Make the app feel like a geometric object rather than a dashboard.

## Non-negotiables
- Square composition inside the R1 portrait viewport
- No browser-app chrome feel
- No loud animation
- No floaty node diagram language
- One primary action only
- Tiny log only

## Layout rules
- Use a tight 2x3 modular composition
- Each tile must feel like a printed panel
- Visual rhythm matters more than literal meaning
- Avoid over-labelling the scene

## Visual rules
- Flat, solid fills
- Strong geometric motifs
- Cropped circle / arc / ring language
- Crisp edges
- Minimal text on the scene
- Composition should read well even without reading labels

## Motion rules
- Sequence the tiles like a composition shift
- No particles
- No blur
- No heavy glow
- Use subtle transform and stroke emphasis only

## UX rules
- R1-first, one thumb, immediate clarity
- Main action centered and obvious
- Log reduced to short trace lines
- Nothing decorative unless it helps structure

## Success criteria
- Looks more like the reference geometric artwork
- Feels premium and disciplined
- Works cleanly on the actual R1 screen
- The screen feels like a designed panel, not an app screen
