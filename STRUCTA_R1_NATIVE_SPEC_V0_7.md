# Structa v0.7 — R1 Native Spec

## Goal
Make Structa feel native, premium, and calm on the Rabbit R1 screen.

## Non-negotiables
- Portrait-first, compact, one-thumb use
- One primary action only
- No desktop chrome, no clutter, no heavy decorative grids
- No wording that leaks implementation intent into the UI
- Tiny, quiet log area only
- Strictly low-motion, high-clarity UI

## Layout rules
- Exact viewport target: R1 portrait-safe small screen
- Top bar: minimal title + status only
- Center: compact six-node hierarchy
- Bottom: short log stack, max 5 visible lines
- Primary button: single full-width pill action

## Graph rules
- 6 total nodes, but hierarchical, not flat
- Core node first
- Supporting nodes reveal in compact cascade
- Lines must be solid, clean, and readable
- Active state must be obvious without glow-heavy effects

## Visual language
- High-contrast dark base
- Clean geometric symbols
- Premium, restrained, symbolic
- No brand words in the UI that distract from the task

## Motion rules
- Subtle scale and line-draw animations only
- No particles
- No blur-heavy effects
- Short animation duration, clear state progression

## UX rules
- One action only
- Immediate feedback
- Minimal text
- Stable structure
- Never overload the tiny screen

## Success criteria
- Feels built for the R1, not adapted to it
- Touchable, legible, and calm
- Looks premium without visual noise
- Easy to test and easy to understand at a glance
